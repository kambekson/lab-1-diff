# Лабораторная работа №1: Учебный DES и дифференциальный анализ

Проект реализует учебный блочный шифр на основе 3-раундовой сети Фейстеля и интерфейс для выполнения дифференциального анализа.

Есть две версии приложения:
- `lab1_des.py` — режим студента (ключ скрыт).
- `lab1_teacher.py` — режим преподавателя (ключ и раундовые ключи показаны).

## Возможности

- Генерация варианта по ФИО и группе (детерминированно).
- Настраиваемые S-блоки `S1`, `S2`, `S3`, таблицы `P` и `EP`.
- Шифрование набора открытых текстов и отображение пар `P`, `P'`, `ΔX/ΔY`.
- Проверка найденного 24-битного ключа (в студенческой версии).
- Вкладка `DDT S-блоков` для вероятностного анализа.
- Экспорт результатов в `.txt`.

## Структура файлов

- `lab1_core.py` — криптологическая логика:
  - шифр `TeachingDES`;
  - DDT-расчеты;
  - подбор лучшей 1-раундовой характеристики;
  - генератор вариантов `VariantGen`.
- `lab1_base.py` — общий GUI и базовая логика для обеих версий.
- `lab1_des.py` — студенское приложение.
- `lab1_teacher.py` — преподавательское приложение.
- `.gitignore` — исключения для git.

## Требования

- Python 3.10+ (рекомендуется Python 3.13).
- `tkinter` (обычно входит в стандартную поставку Python).
- Внешние зависимости не требуются.

## Запуск

Из корня проекта:

```bash
python lab1_des.py
```

или

```bash
python lab1_teacher.py
```

## Как работать (кратко)

1. Откройте нужную версию приложения.
2. На вкладке `Исходные данные` введите ФИО, группу и количество текстов.
3. Нажмите `Сформировать вариант`.
4. При необходимости измените таблицы и дельты, затем запустите шифрование.
5. На вкладке `DDT S-блоков` нажмите кнопку `Сделать анализ`:
   таблицы DDT формируются только по этой кнопке.

## Примечания

- Прерывание приложения через `Ctrl+C` в терминале может дать `KeyboardInterrupt` — это нормальное ручное завершение.
- Для корректного завершения GUI лучше закрывать окно приложения стандартной кнопкой закрытия.
